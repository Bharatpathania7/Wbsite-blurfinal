TO ADD THE PRIVACY IMAGE
========================
1. Download the free Pexels photo from:
   https://images.pexels.com/photos/32174275/pexels-photo-32174275.jpeg?auto=compress&cs=tinysrgb&w=1200
   (free to use, by Selene Madera on Pexels)

2. Save it as:  public/privacy.jpg

That's it. The Privacy section will automatically show it.
Until the file is added, the section shows a blue gradient card with a lock
icon and "100% anonymous" label as a fallback (still looks clean).

For the standalone blur-website.html, put privacy.jpg in the SAME folder as the html.
